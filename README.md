# hoam-design-system

React component library for HOAM.

## Install

```bash
npm install hoam-design-system
```

## Usage

```tsx
import { Button } from 'hoam-design-system';
import 'hoam-design-system/styles.css';

export function Example() {
  return <Button>Shop now</Button>;
}
```

## Peer Dependencies

- `react` `^18 || ^19`
- `react-dom` `^18 || ^19`

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
